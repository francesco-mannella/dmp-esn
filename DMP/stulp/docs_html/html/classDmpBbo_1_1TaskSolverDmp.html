Parameters
[in]dmpThe Dmp to integrate for generating trajectories (that should go through viapoints)
[in]optimize_parametersThe model parameters to change in the Dmp, cf. sec_fa_changing_modelparameters. Depends on the function approximator used for the forcing term.
[in]dtIntegration time steps
[in]integrate_dmp_beyond_tau_factorIf you want to integrate the Dmp for a longer duration than the tau with which it was trained, set this value larger than 1. I.e. integrate_dmp_beyond_tau_factor=1.5 will integrate for 3 seconds, if the original tau of the Dmp was 2.
[in]use_normalized_parameterUse normalized parameters, cf. sec_fa_changing_modelparameters
