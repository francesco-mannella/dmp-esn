Parameters
[in]function_approximatorsThe function approximators for the Dmp (a vector; one element for each Dmp dimension)

[in]taskThe task to optimize (it contains the cost function)
[in]updaterThe parameter updater to use during optimization
[in]directoryDirectory to which to save results
